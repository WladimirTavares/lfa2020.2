\item Considere a seguinte linguagem:

\begin{equation}
L = \{a^ib^jc^k | i = j \mbox{ or } i = k\}  
\end{equation}

\begin{enumerate}
 \item Projete uma gramática livre de contexto para $L$.
 \item Projete um autômato de pilha para $L$.
 \item Escolha uma string $w \in L$ com $|w| \geq 6$
 \item Mostre uma derivação mais à esquerda para $w$.
 \item Construa uma árvore sintática para $w$.
 \item Encontre uma string $w \in L$ com $|w| \geq 6$ com duas árvores sintáticas diferentes.
 \item Escolha uma string $w \in L$ com $|w| \geq 8$. Decomponha $w$ em $u,v,x,y,z$ tal que $|vxy| \leq 8$ e $|vy| > 0$ e $uv^ixy^iz \in L$ para todo $i \geq 0$
\end{enumerate}


\item Considere a seguinte gramática:

\begin{center}
\begin{tabular}{lll}
S & $\to$ & $aSb$\\
S & $\to$ & $Sb$\\
S & $\to$ & $\varepsilon$\\
\end{tabular}
\end{center}

\begin{enumerate}
 \item Liste 4 palavras geradas por essa gramática. 
 \item Dê uma descrição informal da linguagem gerada por essa gramática.
 \item Mostre que essa gramática é ambígua apresentando uma string que possui duas árvores sintáticas diferentes. Desenhe as 
 duas árvores sintáticas.
 \item Dê uma gramática não ambígua que aceita a mesma linguagem da gramática acima. Explique sua resposta.
 \item Escolha uma string $w \in L$ com $|w| \geq 8$. Decomponha $w$ em $u,v,x,y,z$ tal que $|vxy| \leq 8$ e $|vy| > 0$ e $uv^ixy^iz \in L$ para todo $i \geq 0$
\end{enumerate}


\item Considere a seguinte linguagem:

\begin{equation}
L = \{a^nb^m | n \leq m \leq 2n\}  
\end{equation}


\begin{enumerate}
\item Projete uma gramática livre de contexto para $L$.
\item Encontre uma string $w \in L$ com $|w| \geq 5$ com duas árvores sintáticas diferentes.
\item Dê uma gramática não ambígua que aceita a mesma linguagem da gramática acima. Explique sua resposta.
\item Elimine todas as $\varepsilon$-regras da gramática do item (a) (Transformação DEL).
\item Elimine os símbolos terminais das regras com o tamanho do lado direito maior que 1 (Transformação TERM)
\item Reduza para regras com o tamanho do lado direito menor ou igual a 2 (Transformação BIN).
\item Execute o algoritmo CYK com a string do item (b). Explique como o algoritmo CYK pode ser modificado para identificar se a gramática é ambígua.
\end{enumerate}


\item O \textbf{lema do bombeamento para linguagens livre de contexto} pode ser entendido como um jogo. 
Suponha que seu oponente acredita que $L$ não é livre de contexto e você acredita que a linguagem é livre de contexto. Você escolhe um valor de $p$. 
Depois, seu oponente escolhe uma string $w$ tal que $|w| \geq p$. Você escolhe $u,v,x,y,z$ tal que $w = uvxyz$ satisfazendo 
$|vxy| \leq p$, $|vy| \geq 1$. Se o seu oponente escolhe um valor de $i \geq 0$ tal que $uv^ixy^iz \not\in L$, ele ganha 
Caso contrário, você ganha. 

A linguagem escolhida foi $L = \{ww | w \in \{0,1\}^*\}$ (L não é livre de contexto, ou seja, seu oponente vai ganhar se fizer a escolha errada).
Você escolheu $p = 3$ e seu oponente escolhe $w = baabaa$.

\begin{enumerate}
 \item Encontre uma decomposição de $w$ em $u,v,x,y$ e $z$ tal que para todo $i \geq 0$, $uv^ixy^iz \in L$?
 \item Se você tivesse escolhido $p=2$ e seu oponente escolhido o mesmo $w$, você poderia ganhar o jogo?
 \item Para $p=3$, qual é a string que seu oponente deveria ter escolhido para vencer o jogo?
\end{enumerate}



\end{enumerate}
